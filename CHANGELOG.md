# Changelog

All notable changes to AirOps RP Weather are documented in this file.

## [1.2.1] - 2026-07-24

### Changed

- Made the flight-condition status line more compact.
- Replaced abbreviated wind direction with a written direction.
- Displayed wind gusts separately from sustained wind.
- Renamed and simplified the weather-development section.
- Improved readability of the hourly forecast on mobile devices.
- Reduced symbol density in the forecast.
- Cleaned up the embed footer.
- Updated the README for Railway deployment using `python3 main.py`.

### Fixed

- Prevented mixed presentation between the previous and revised embed layouts.

## [1.2.0] - 2026-07-24

### Added

- Anticipatory flight assessment for the next 60 minutes.
- Operational weather hints.
- Weather-development analysis.

### Changed

- Improved mobile layout.
- Improved hourly forecast formatting.
- Improved general embed presentation.

## [1.1.0]

### Added

- Flight-condition assessment.
- Weather trend evaluation.

### Changed

- Improved embed appearance and internal processing.

## [1.0.0]

### Added

- Initial Discord integration.
- Open-Meteo integration.
- Automatic weather updates.
- Persistent weather embed.
- Railway deployment.
